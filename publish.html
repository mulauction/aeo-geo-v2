Amplify - 콘텐츠 배포 및 실행

Amplify는 생성된 콘텐츠를 배포하고 실행 로그를 기록합니다. 배포/실행은 점수에 영향을 주지 않습니다.
현재는 자동 등록이 아니라, 생성된 HTML을 복사해 붙이는 방식입니다.

A. Webflow(권장) 5단계 가이드

  1. Generate에서 HTML 생성
  2. output 전체 복사
  3. Webflow 프로젝트 열기
  4. 대상 페이지에 Embed/HTML 블록 추가
  5. 붙여넣기 → Publish

B. 스마트스토어/쿠팡준비중

플랫폼 편집기 제약으로 자동화는 Phase 3 이후에 제공 예정입니다.

현재는 가이드 문서 제공 예정입니다.

C. 체크리스트

  • 복사한 블록이 <div>로 시작/끝나는지
  • 불필요한 스크립트가 없는지
  • FAQ가 포함됐는지(선택)
